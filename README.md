# Similar

## Objective

This react program compares 2 JSON objects and gives a score from 0 to 1 depending on how similar are they.

1 depicts identical.
0 states dissimilar.

## Stack

1. ReactJS
2. Javascript
3. Enzyme for Testing

## How To Run Locally

1. git clone <git url>
2. yarn install
3. yarn start
